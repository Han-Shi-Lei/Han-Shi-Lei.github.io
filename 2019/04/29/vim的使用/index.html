vim的使用

2019-04-29 Linux学习

vim [文件]：打开一个文件，不存在就会创建新文件

:q 退出

x 删除光标下的字符

i 进入插入模式，光标在当前位置之前

A 进入插入模式，光标在行位

dw 从当前位置

de 从当前位置删除到单词末尾

d$ 从当前位置删除到行末

dd 删除一整行

ctrl+z 切换到后台

fg 切换回刚才的程序

u 撤销一处更改 undo

U 撤销整行的更改

ctrl+r redo

vim 中删除的内容都放在剪贴板（寄存器）中

p 黏贴剪贴板中的内容

r 替换光标位置的字符

c 修改类命令 删除内容的同时进入插入模式

ctrl + g显示文件状态信息

G 来到文件末尾

[number]G跳转到任意指定行

：[number] 跳转到任意指定行

gg 来到文件开头

“ 回到上次的位置

/ 进行字符串查找n是查找下一个，N查找上一个

% 查找到匹配的括号

:! 这是在vim外部执行一个命令

o 在当前行的下方创建一个新行，并进入插入模式

O 在当前行的上方创建一个新行，并进入插入模式

i 在当前光标的之前进入插入模式

I 在行首进行插入模式

a 是在当前光标之后进入插入模式

A 是在行末进行插入模式

R 进入替换模式

v 进入可视模式（选中一段内容）

y 复制选中的内容

p 粘贴剪贴板的内容

：set设置一些编辑器选项

：set monse = a启用鼠标

：set num 启用行号

：set hlsearch高亮搜索结果

ctrl + s：xshell 冻结当前界面

ctrl+q： 解除冻结
